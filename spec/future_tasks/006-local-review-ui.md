# Future Task: Local Review UI

## Goal

Build a simple interface for reviewing postings, updating preferences, and marking roles as interesting or not relevant.

## Requirements

- Show collected and filtered postings.
- Allow the user to mark postings as interested, applied, ignored, or needs follow-up.
- Allow editing search preferences without manually changing Markdown files.
- Keep the first version simple, likely a local web app or lightweight dashboard.

## Output

- A local UI for reviewing and managing internship search results.
