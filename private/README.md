# Private Local Files

Use this directory for personal files that should not be committed to git.

Suggested local files:

- `profile.md` for interests, skills, classes, projects, and preferences.
- `resume.md` or `resume.pdf` for resume details.
- `companies.txt` for target companies.
- `job_sites.txt` for job boards and career portals.

Everything in this directory is ignored by git except this README and `.gitkeep`.
